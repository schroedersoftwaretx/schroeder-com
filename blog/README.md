# schroeder.com

Personal site: portfolio, resume, and blog (Next.js App Router, MDX, Tailwind).

## Local development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).


## Production URL

Canonical site URL and metadata use `NEXT_PUBLIC_SITE_URL` when set (no trailing slash). Defaults are defined in `app/site.ts`.

## Build

```bash
pnpm build
pnpm start
```
